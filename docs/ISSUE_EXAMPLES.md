# Issue Examples / 工单示例

## P0 Example / P0 示例
- **Title:** [P0] Checkout failure in production / 生产下单失败
- **Summary:** Production checkout is blocked for all tenants. / 生产环境所有租户无法下单。
- **Impact:** Revenue loss, must rollback. / 收入损失，必须回滚。

- **Title:** [P1] Invoice PDF generation slow / 发票 PDF 生成过慢
- **Summary:** 50% of invoices take >30s to generate. / 50% 发票生成超过30秒。
- **Impact:** Business operations degraded, needs fix. / 业务受影响，需要修复。
